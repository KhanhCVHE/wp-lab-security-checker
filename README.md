# wp-lab-security-checker

A lightweight open-source CLI for defensive WordPress surface auditing in authorized environments.

## Features
- Detect exposed WordPress metadata
- Check common public endpoints
- Identify basic exposure and misconfiguration patterns
- Export reports in JSON and Markdown

## Who this is for
This tool is intended for maintainers, students, defenders, and small teams auditing WordPress installations they own or are explicitly authorized to test.

## Safety
This project is for defensive and authorized security review only.
